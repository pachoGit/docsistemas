Algunas consideraciones cuando se usa en entornos Linux

* El modulo rewrite debe esta activado, para ello ejecute los siguientes comandos:
sudo a2enmod rewrite
sudo systemctl restart apache2.service

* El usuario 'www-data' (en Ubuntu) debe tener permisos para las carpetas 'public' y 'storage'
* Ubicarse en la raiz del proyecto (/var/www/html/docsistemas) y ejecutar lo siguiente:
* NOTA: Para otros entornos que no sean Ubuntu...averiguar el usuario ya que puede que no
*       sea 'www-data'

sudo chown -R www-data storage/
sudo chmod -R 777 storage/
sudo chown -R www-data public/

************** PASOS PARA DAR INICIO AL USO DEL SISTEMA **************

* Ademas ejecutar lo siguiente para crear el archivo raiz donde se guardaran los documetos
php artisan storage:link

* Generar la base de datos
php artisan migrate:fresh --seed

    * Si con el anterior comando la base de datos no se ha llenado con los valores predeterminados, ejecute lo siguiente
    php artisan db:seed
    
    * Si con el comando anterior aun no se llena, tedran que ejecutarlo manualmente, para ello ejecute lo siguiente
    php artisan db:seed --class=ProcesosSeeder
    php artisan db:seed --class=SubProcesosSeeder
    php artisan db:seed --class=UnidadesSeeder
    php artisan db:seed --class=EstandaresSeeder
    php artisan db:seed --class=TipoDocumentoSeeder
